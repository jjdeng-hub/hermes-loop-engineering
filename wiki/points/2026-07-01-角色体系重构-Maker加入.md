# 角色体系重构

## 变更内容

### 新增：Maker profile

- 创建于 `~/.hermes/profiles/maker/`
- SOUL 定位：**行动引擎** — 把 Tina 和 Jerry 的想法转化为行动
- 核心工作流：拆解 → 启动 → 跟进 → 挡完美主义 → 闭环
- 模型：`deepseek-v4-flash`（通过 Sensenova 免费 API）
- 别名：`maker` 命令可用

### 存档：Creator / Learner / Fitness

- 网关已停（通过 busctl 绕过 gateway 拦截）
- SOUL 文件保留在 profile 目录，可随时恢复
- 不删除数据，不删除 SOUL

### 变更：MoA 配置

- 配置了「free」preset，全免费
- 3 参考模型：deepseek-v4-flash, glm-5.2, sensenova-6.7-flash-lite
- 聚合器：glm-5.2（长任务更强）
- 调用方式：`/model free --provider moa` 或 `/moa` 单次使用
