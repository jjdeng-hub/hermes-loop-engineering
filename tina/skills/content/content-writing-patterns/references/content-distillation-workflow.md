# 内容创作者蒸馏工作流

> 将目标内容创作者的五层认知蒸馏成可复用的结构模板

## 使用前提

- 搜索功能可用（YouTube搜索、知乎搜索等）
- 或浏览器可用（直接浏览目标创作者的内容）
- 已安装 `xiaozheng-nuwa` 技能

## 蒸馏流程

### Step 1: 确定目标

用户说「蒸馏一个XXX」或用户想「抄」某个创作者。

### Step 2: 六路采集（参考 xiaozheng-nuwa Phase 1）

尽量覆盖：
1. 该创作者的视频/文章列表（YouTube频道、知乎专栏、公众号）
2. 爆款内容分析（哪些内容互动最高）
3. 社交媒体表达（微博、小红书上的发言风格）
4. 批评者视角（争议点、翻车案例——反模式）
5. 决策记录（为什么做这个内容、转型节点）
6. 时间线（起步→增长→成熟）

### Step 3: 五层认知框架（xiaozheng-nuwa L1-L5）

| 层次 | 分析内容 | 产出 |
|------|---------|------|
| L1 表达DNA | 语气(口语化/正式/朋友式)、节奏(快/慢)、用词偏好(叹词/技术词/网络梗) | 一段示范对话 |
| L2 心智模型 | 3-5个核心认知框架（如：门槛即流量、工具即视角） | 每个模型+案例 |
| L3 决策启发式 | 5-10条创作决策规则（如：开头3秒要有视觉钩子） | 可直接使用的规则 |
| L4 反模式 | 3-5条不做什么（如：不做大而全教程、不写纯理论） | 禁止清单 |
| L5 诚实边界 | 2-3条做不到什么 | 明确局限 |

### Step 4: 提示模板

```markdown
# [创作者名] 内容结构蒸馏

## L1 表达DNA
[语气/节奏/用词 + 一段示范]

## L2 心智模型
### 模型1：[名称]
- 核心：一句话
- 案例：[2-3个内容中的体现]

## L3 决策启发式
1. [规则1]
2. [规则2]
...

## L4 反模式
- 不做什么...
- 价值观底线...

## L5 诚实边界
- 做不到什么...

## 对你最有用的
[这个创作者最值得「抄」哪个结构/模型]
```

### Step 5: 可复用的结构模板输出

最终输出应该是 Jerry 可以直接用的模板——告诉他「这个人的结构是这样，你用自己的话套进去」。

## ⚠️ 注意事项

- 蒸馏的是「认知操作系统」，不是复读语录
- 所有内容必须基于公开信息，不编造
- 三重验证：同一观点在≥2个不同场景中出现过 → 能推断对新问题的立场 → 不是所有聪明人的共识
- 信息不足时明确标注「公开资料有限，谨慎采用」
