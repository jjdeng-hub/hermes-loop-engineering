# 抖音封面提示词

用于豆包/即梦等 AI 生图工具生成抖音竖版封面（9:16）。

## 风格指南

- AI 话题：科幻/概念艺术风格，深色背景 + 霓虹色点缀
- 个人反思类：极简、留白多、低饱和度
- 观点分析类：几何/结构化构图，蓝紫冷色调

## 核心规则

- 封面上不放标题文字（平台会自动叠加）
- 构图偏上 1/3，为平台标题留空间
- 不要太复杂——抖音封面在信息流中很小

## 示例 Prompt

```
9:16 vertical poster, dark sci-fi concept art style, 
abstract neural network glowing in deep blue, 
neon cyan accents, minimalist composition, 
no text, clean negative space in upper third,
trending on artstation style
```
