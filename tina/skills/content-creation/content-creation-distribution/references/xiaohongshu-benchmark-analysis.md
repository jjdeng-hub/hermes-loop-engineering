## 二、分析维度

### 1. 赞藏比（内容类型信号）
| 内容类型 | 赞藏比典型值 | 含义 |
|----------|------------|------|
| 教程类 | 0.6-0.8 | 收藏 > 点赞 = 动手价值高 |
| 观点/热点类 | 1.5-8.0 | 评论爆炸，讨论驱动 |
| 工具分享类 | 0.4-0.6 | 高收藏价值，工具型内容 |

### 2. 封面风格分析
| 账号 | 封面风格 | 效果 |
|------|---------|------|
| 库森 | 暗色底 + 大字 + 代码截图 | 教程类最佳 |
| 阿亮 | 暗色底 + 工具截图 | 工具分享类 |
| 卡兹克 | 暗色底 + 醒目标题 + 事件截图 | 热点/观点类 |

**结论**：所有高互动笔记封面均为暗色底。暗色封面点击率比亮色底高 30-50%。

## 三、模板化应用

分析对标后，为你的内容提取可复用框架：

### 教程类模板
```
标题：[工具名] + [能做什么/怎么用]
正文：
  1. 钩子句（具体痛点场景）
  2. 工具介绍（一句话说清能干嘛）
  3. 操作步骤（3-5步，截图+标注）
  4. 效果展示（前后对比）
  5. 引导互动
配图：4-6张截图，红框标注关键步骤
```

### 观点/热点类模板
```
标题：[事件] + [你的态度/解读]
正文：
  1. 发生了什么事
  2. 为什么重要（独家解读）
  3. 对读者的影响
  4. 引导讨论
配图：2-4张事件截图 + 1张你的观点
```

### 工具分享类模板
```
标题：[一个/找到] + [形容词] + [做什么] + [工具名]
正文：
  1. 发现了什么（勾起好奇）
  2. 能做什么（场景化描述）
  3. 怎么用（简单几步）
  4. 价值总结
配图：3-5张工具截图 + 效果展示
```
