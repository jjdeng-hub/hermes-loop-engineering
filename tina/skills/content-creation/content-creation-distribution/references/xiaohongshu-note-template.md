### 正文文案

```
Hermes Agent 最近大更新，出了一个 Web 管理面板叫 Dashboard。

以前管理 AI Agent 全靠敲命令：
sessions list  —— 看历史记录
cron list      —— 管理定时任务
config edit    —— 改配置文件

现在打开浏览器，输入 localhost:9119，所有功能都在一个页面上。

我用了之后觉得最实用的几个功能：

📁 Sessions —— 所有历史会话都在这里，按时间排列，还能搜索。
⏰ Cron —— 定时任务管理，可以设每天早上自动跑的任务。
⚙️ Config —— 浏览器里改配置，不用 vim，不用担心缩进。
🔄 Restart Gateway —— 一键重启，不用敲命令。
📦 Update Hermes —— 一键更新到最新版。

支持中文界面和深色/浅色主题切换。

怎么用？装了 Hermes Agent 的话：
hermes dashboard
浏览器自动打开，不需要任何额外配置。

还没用过的可以去 GitHub 搜 Hermes Agent，14 万 Stars，开源免费。

觉得有用的话收藏一下，下次用的时候直接翻出来～
```
