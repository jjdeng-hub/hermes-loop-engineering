# 招聘市场需求调研方法

## 为什么看招聘网站

招聘需求是市场最诚实的信号。企业愿意为技能付多少工资 = 市场的真实定价。

## 工具与技巧

### 数据源
- **猎聘** (liepin.com) — 中高端岗位，薪资透明
- **51job** (51job.com) — 覆盖广，小公司多
- **Boss直聘** (zhipin.com) — 反爬严格，不一定能用

### 关键搜索词
| 搜索词 | 能发现什么 |
|---|---|
| AI Agent | Agent 开发/产品/架构岗位 |
| LangChain + RAG | 具体技术栈需求 |
| 大模型应用 | 应用层岗位（非算法） |
| 数字员工 | 企业自动化方向 |
| AI + [行业] | 垂直行业需求（如 AI 制造） |

### 数据提取方法
使用 browser_console 提取页面上的职位数据：
```javascript
Array.from(document.querySelectorAll('[class*="job"]'))
  .map(el => el.innerText.trim())
  .filter(t => t.length > 20)
  .join('\n---\n')
```

### 分析维度
1. **岗位类型分布** — 开发/产品/算法/管理的比例
2. **薪资区间** — 按经验和城市分层
3. **高频技能词** — 哪些技术栈出现最频繁
4. **行业分布** — 哪些行业在招 AI 人才
5. **经验要求** — 门槛是 1-3 年还是 5-10 年

## 典型发现（2026.06 北京+苏州）

| 岗位类型 | 薪资范围 |
|---|---|
| Agent 开发工程师 | 12K-40K/月 |
| Agent 产品经理 | 18K-50K/月 |
| Agent 架构/算法 | 35K-100K/月 |

高频技能：Python > RAG > LangChain > Prompt Engineering > 向量数据库 > API 设计
