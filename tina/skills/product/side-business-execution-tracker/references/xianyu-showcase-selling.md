# 闲鱼「展示先行」卖法

## 核心原则

**先做出能让人一眼看到效果的东西，再挂链接。** 不跟客户描述"我能做什么"——直接让他看到。

## 闲鱼平台规则

### 敏感词（会触发警告/封禁）
- ❌ 「代写」（学术不端）
- ❌ 「代做」「代考」
- ❌ 「刷」「量」「粉」

### 安全替代词
- ✅ 「辅助」「协助」「优化」「美化」「改造」
- ✅ 用「设计」「排版」「实现」替代「代写」
- ✅ 用「界面改造」「效果图」替代「代做页面」

## 展示页制作公式

### 目标客户 × 页面类型

| 客户 | 做什么页面 | 展示什么 |
|---|---|---|
| 独立开发者 | SaaS Landing | 暗色·玻璃态·金色，像 Linear |
| 做后台的程序员 | Dashboard | 洁净白色·数据卡片·侧边栏 |
| AI/App 创业者 | App 展示 | 霓虹科技·手机模拟·渐变 |
| 接私活的设计师 | Portfolio | 杂志感·衬线体·作品网格 |
| 职场/学生 | PPT 美化 | Before/After 同屏对比 |

### 页面要求
- 单文件 HTML，无外部依赖（不需要服务器）
- 用户双击直接用 Chrome 打开
- Chrome F11 全屏 → Ctrl+Shift+P → "Capture full size screenshot"
- 每个页面风格必须不同，展示能力广度

## 定价策略

| 服务 | 起步价 | 交付时间 | 说明 |
|---|---|---|---|
| 网页/APP 界面改造 | ¥30 | 3h 内出图 | 客户发截图 → 出改造效果图 |
| PPT 排版美化 | ¥30 | 3h 内出图 | 客户发原稿 → 出美化版 |
| 前端页面实现 | ¥50 | 当天 | 设计稿 → HTML/CSS |
| AI 辅助代码 | ¥50-200 | 沟通确定 | Python/前端小需求 |

## 成交信号

- 有人问「能不能帮我做 XXX」→ 先出一个效果图，觉得好再付
- 有人已读不回 → 正常，不是失败
- 有人说太贵 → 不降价，可以缩小交付范围

## 已验证的有效策略

1. **先出图后付钱**：降低客户决策门槛
2. **48 小时内必有结果**：挂上后两天内没人问 → 调整描述或降价
3. **不要只挂一个服务**：网页改造 + PPT 美化 + 代码实现同时挂
4. **简介去敏感化**：所有词都要通过平台审核
