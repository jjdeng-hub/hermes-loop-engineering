# 文件传输模式（公司 → 家庭 RAG 系统）

## 场景

公司电脑和家庭电脑独立，无法直接传输文件。需要在公司整理的文件流入家庭 RAG 系统。

## 推荐方案：Outlook 邮件桥

```
公司电脑                      家庭电脑
   │                             │
   ├─ 整理笔记/导出 PDF           │
   ├─ Outlook 发邮件给自己 ──────→ 收件箱
   │                             ├─ 下载附件/复制正文
   │                             ├─ 保存到 桌面\rag-data\
   │                             └─ 30s 后自动索引 ✅
```

### 优势
- Outlook 企业标配，无需额外软件
- 邮件天然是备份
- 附件 + 正文都支持
- 不涉及 U 盘或微信传输的安全风险

## Symlink 桥接（WSL）

让 Windows 桌面文件自动流入 WSL 的 RAG 监控目录：

```bash
ln -s /mnt/c/Users/jjdeng/Desktop/rag-data ~/rag-agent/data/company-docs
```

⚠️ 扫描器必须用 `rglob('*')` 而非 `iterdir()` 才能跟踪 symlink 目录。

## 限制

| 限制 | 说明 |
|------|------|
| 公司合规 | 确认文件可外发（非机密技术文档通常 OK） |
| PDF 扫描件 | pypdf 只读文字型 PDF，扫描件需先 OCR |
| 附件大小 | 邮件通常限 25MB |
| 传输延迟 | 非实时，依赖手动发邮件 + 手动下载 |
