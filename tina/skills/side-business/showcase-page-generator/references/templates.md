# 四个已验证展示页面模板

以下四个模板已经过视觉验证，每个风格完全不同。使用时直接复制完整 HTML 到目标文件。

## 1. 暗色奢华 SaaS Landing

- 风格：Linear 风格，黑底金色，玻璃态卡片
- 适用：工具型 SaaS、生产力产品
- 特色：玻璃态毛玻璃卡片、金色点缀、三层定价、数据指标展示

## 2. 洁净瑞士风 Dashboard

- 风格：白底建筑感，极简，瑞士平面设计风格
- 适用：B2B 后台、数据分析工具
- 特色：侧边栏导航、KPI 卡片、柱状图（纯 CSS）、交易流水

## 3. 霓虹科技 App 展示

- 风格：全黑底、青-紫渐变、发光效果
- 适用：AI 产品、消费级 App
- 特色：手机模型+聊天模拟、霓虹渐变字体、技术规格展示

## 4. 编辑风 Portfolio

- 风格：奶白底、衬线体、杂志排版
- 适用：设计工作室、品牌服务
- 特色：超大衬线标题、四宫格作品集、客户引言、古典金色点缀

---

生成命令示例：

```
在 /mnt/c/Users/jjdeng/Desktop/showcase-pages/ 下生成 4 个独立 HTML 文件，
每个文件使用不同的设计风格：
1. 暗色 SaaS landing page
2. 白底 Dashboard
3. 霓虹 App 展示
4. 编辑风 Portfolio

要求：单文件、纯内联 CSS、零 CDN 依赖、桌面端可双击打开截图。
```
