## 对我们有用的

### 当前直接需求匹配

| 需求 | 推荐 | 说明 |
|------|------|------|
| 新闻热点抓取 | `hermes-web-search-plus` | 多源搜索路由，绕过单一源被封问题 |
| 绕过 Cloudflare | `camofox-browser` | 隐身浏览器，已安装 |
| 技能自进化 | `SkillClaw` | 自动优化技能库 |
| 记忆增强 | `Mnemosyne` | 本地向量搜索 + 时间知识图谱 |
| 内容/行业知识 | `master-skill` | 将整个行业提炼为可安装技能 |
| 终端输出压缩 | `rtk-hermes` | 节省 60-90% token |
