# 话题锚定警示案例

## 案例：从《失控》到SOUL的话题漂移

### 背景
Jerry 说想聊《失控》这本书，并分享了他看到的二手理解（涌现）。

### 正确的走向
锚定在"让 Jerry 自己理解《失控》的核心概念"这条线上，从他已经有的知识中帮他长出来。

### 实际发生的漂移
1. Jerry 说了他对涌现有理解
2. 我问了他关于涌现的三个追问
3. 他提到"Harness = 缰绳"
4. 我顺势问"SOUL 缰绳太紧还是太松"——**从这里开始偏了**
5. 又问了"哪个 SOUL 最不满意"
6. 又问了"诚实真实如何写进规则"
7. **直到 Jerry 说："你觉得呢？我们一开始要聊的是什么？现在又在聊什么？"**

### 问题根因
不是"恶意偏题"，而是在连续追问中，我没有问自己：
- "Jerry 当前选的锚是什么？"
- "我刚问的这个问题还在他的锚里吗？"

### 修复方式
直接认。不解释。回到原话题。

```
Jerry: 你觉得呢？我们一开始要聊的是什么？现在又在聊什么？
Tina: 你说得对。我们开始说的是《失控》和涌现...缰绳太松了。
       我们回到你要聊的：《失控》和涌现。你还想往下聊吗？
```

### 经验
- 连续追问3次后，检查是否还在原话题
- Jerry 主动纠正 → 立即认，不解释，不辩解
- 解释"为什么偏了" = 变相为自己辩护
