# Assessment Question Bank

These questions are battle-tested. Use them verbatim, one batch at a time (2-3 questions per batch).

## Batch 1: Physical Baseline

"先摸身体底：你的身高体重？最近一次运动是什么时候、做了什么？每天大概睡几小时、质量如何（入睡困难/易醒/睡不醒）？"

## Batch 2: Cognitive Input

"最近三个月，有没有一本让你觉得「读晚了」的书或一篇文章？你平时获取新信息的主要渠道是什么（抖音/公众号/播客/书/其他）？"

## Batch 3: Emotional State

"最近一个月，有没有一件让你明显焦虑或烦躁的事？反过来，有没有一件事让你觉得「今天没白过」？"

## Batch 4: Daily Schedule + Vision

"现在工作日从起床到睡觉，大致的时间线是怎样的？另外——如果三年后的你回头看现在的自己，你最希望那时候的你「庆幸自己今天做了什么」？"

## Schedule Review Follow-up

After mapping the daily timeline, explicitly ask about unmentioned commitments:
"除了工作和通勤，你现在每天还有哪些固定要做的事？比如内容创作、口播练习——这些需要多少时间？"

## Plan Feedback

After presenting the plan:
"这个方案可以落地吗？有什么需要调整的？"

Options to offer:
1. 可以，今天就开始
2. 动作有点多，能再简化
3. 我先调整一下再开始
4. 帮我把这个写成文件存起来
