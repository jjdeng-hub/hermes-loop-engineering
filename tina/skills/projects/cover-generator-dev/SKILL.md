# 小红书封面生成器 - 开发指南

## 项目状态
已完成MVP开发，用户正在测试

## 技术方案
- 纯前端：HTML5 + CSS + JavaScript + Canvas
- 图片在浏览器端生成（所见即所得）
- 部署：Vercel（免费）

## 封面模板配置（9套）
### 赛道1：职场干货（深色专业感）
- 模板A：深蓝渐变背景，白色标题，红色细线装饰
- 模板B：纯黑背景，数字放大突出，绿色侧边
- 模板C：深灰背景，白色标题，右侧图标

### 赛道2：知识干货（白底+重点色）
- 模板A：纯白背景，黑字，左侧红色色块
- 模板B：浅灰背景，关键词高亮，圆角卡片
- 模板C：白底，描边文字，数据对比

### 赛道3：成长自律（温暖清新风）
- 模板A：浅粉渐变，深粉色标题，手绘图标
- 模板B：浅绿背景，绿色标题，植物图标
- 模板C：奶油白背景，橙色标题，温暖元素

## 用户流程
1. 输入标题（必填，最多50字）
2. 输入正文摘要（选填，最多50字）
3. 选择赛道（职场/知识/成长）
4. 点击「生成封面」
5. 预览3套方案，点击选中
6. 导出PNG（1080×1080px）
